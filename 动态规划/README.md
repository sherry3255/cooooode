# 算法模板

- 动态规划问题的一般形式是求最值（最长递增子序列、最小编辑距离）；
- 动态规划问题的核心是穷举，因此动态规划的三要素为：
    - 重叠子问题
    - 最优子结构
    - 状态转移方程
- 状态转移方程思维框架
    - 明确 base case -> 明确「状态」-> 明确「选择」 -> 定义 dp 数组/函数的含义。

```
# 初始化 base case
dp[0][0][...] = base
# 进行状态转移
for 状态1 in 状态1的所有取值：
    for 状态2 in 状态2的所有取值：
        for ...
            dp[状态1][状态2][...] = 求最值(选择1，选择2...)
```
